# EnterpriseMonitoring

EnterpriseMonitoring is a source-controlled infrastructure automation project for building and managing an enterprise monitoring platform.

The primary product within this repository is **TZMonitor**, a rebranded monitoring solution based on **Zabbix 7.0 LTS**, built entirely from source and deployed using Ansible.

## Project Goals

- Fully automated infrastructure deployment
- Source-based compilation of Zabbix 7.0
- Enterprise-grade Ansible automation
- Modular and reusable role design
- Secure configuration using Ansible Vault
- Professional Git workflow and documentation

## Technology Stack

- Rocky Linux 9.8
- Ansible Core 2.14
- PostgreSQL 16
- Nginx
- PHP-FPM
- Git & GitHub
- Visual Studio Code
- Oracle VirtualBox

## Repository Structure

- `ansible/` - Infrastructure automation
- `branding/` - TZMonitor branding assets
- `docs/` - Technical documentation
- `scripts/` - Utility scripts
- `tests/` - Validation and testing
- `source/` - Source archives and patches
